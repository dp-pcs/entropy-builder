# Entropy Brand System

**Location**: `/entropy-brand-assets/`

## Quick Reference

- **Logo**: Organic network mark (9-12 asymmetric nodes)
- **Primary Color**: Charcoal `#2B2D42`
- **Accent Color**: Sage Green `#4A6B5C`
- **Typography**: Space Grotesk (headings), Inter (body)
- **Theme**: Light backgrounds, warm neutrals, professional minimal

## For AI: Brand Application Instructions

When creating designs or interfaces for Entropy:

1. **Colors**: Use soft white (#F8F9FA) or warm off-white (#FAF7F2) backgrounds
2. **Logo**: Use horizontal lockup for wide spaces, stacked for squares
3. **Network aesthetic**: Organic, asymmetric node patterns (not rigid geometry)
4. **Typography**: Space Grotesk Bold for headlines, Inter for body text
5. **Tone**: Professional, calm, intelligent, boardroom-ready

See `/entropy-brand-assets/README.md` for complete specifications.
